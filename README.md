# A World Away - Exoplanet Detection with AI

An interactive web application for detecting exoplanets using machine learning, featuring a 3D solar system visualization and enhanced starfield background.

- **Discovery Cards**: Interactive cards for confirmed exoplanets

## Technology Stack

- **Frontend**: React 18, TypeScript, Vite
- **3D Graphics**: React Three Fiber, Three.js
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Icons**: Lucide React

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

## Components

- `EnhancedStarField`: Interactive starfield with mouse effects
- `SolarSystem`: 3D rotating solar system
- `MLModelTraining`: Machine learning model training interface
- `ExoplanetDiscoveryCard`: Individual exoplanet information cards
- `LightCurveChart`: Data visualization for light curves
- `FeatureImportance`: SHAP analysis visualization

## NASA Space Apps Challenge 2024

This project was developed for the NASA Space Apps Challenge 2024, focusing on exoplanet detection and analysis using artificial intelligence.
